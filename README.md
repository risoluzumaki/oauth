This repository contains implementations of an OAuth REST API using various backend technologies.

## API Documentation

This project uses the OpenAPI specification for API documentation.

-   **Specification File**: You can find the full API definition in the [`./docs/openapi.yaml`](./docs/openapi.yaml) file.
-   **Viewing the Documentation**: To view the documentation in a user-friendly UI, you can use tools like the [Swagger Editor](https://editor.swagger.io/) or [Redocly](https://redocly.github.io/redoc/). Simply paste the content of `docs/openapi.yaml` into the editor.
